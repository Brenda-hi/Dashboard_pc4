02

Identificación de la carta de control

La elección de la carta depende del tipo de dato (variable continua o atributo), del tamaño de subgrupo y de qué se quiere monitorear: la tendencia central, la dispersión, o la proporción/conteo de defectos.

Tabla de referencia — selección de carta de control
Tipo de dato Tamaño de subgrupo Carta recomendada Qué monitorea
Variable continua n = 1 I-MR (Individuales y rango móvil) Valor individual y variabilidad punto a punto
Variable continua 2 ≤ n ≤ 10 X̄-R (Medias y rangos) Tendencia central y dispersión (rango)
Variable continua n > 10 X̄-S (Medias y desviación estándar) Tendencia central y dispersión (desv. estándar)
Atributo (defectuoso / no defectuoso) n variable p Proporción de unidades defectuosas
Atributo (defectuoso / no defectuoso) n constante np Número de unidades defectuosas
Atributo (número de defectos) n variable u Defectos por unidad
Atributo (número de defectos) n constante c Número total de defectos por unidad de inspección

04

Desarrollo en Minitab

Ver minitab/README.md para la guía paso a paso y la plantilla de datos lista para importar.

  1. Importar los datos: File → Open Worksheet o pegar directamente en la hoja de trabajo, una columna por posición dentro del subgrupo (o una columna de datos + una columna de subgrupo).
  2. Ir a Stat → Control Charts → Variables Charts for Subgroups → Xbar-R… (ajustar según la carta identificada en la sección 02).
  3. Seleccionar las columnas de datos y el tamaño de subgrupo.
  4. En Xbar-R Options, revisar las pruebas de causas especiales (Tests) que se desean aplicar.
  5. Ejecutar y exportar el gráfico como imagen.
  6. Si se usa Minitab Assistant: Assistant → Control Charts → Xbar-R Chart, seguir el asistente y revisar el Report Card generado.
